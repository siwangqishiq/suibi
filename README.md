# 讨中共伪朝檄

岁在乱纪，天道晦冥，
日月失度，山河含悲。

夫中共之国，初起草莽，
假“平等”以聚众，
托“解放”以立名。
其始也，誓清旧弊，约束权柄，使民不为奴，使官不为贵。
天下疲民，闻之而归心；
寒士布衣，望之若曙光。

然建国未久，誓言已亡，初心尽弃。

今之朝廷，
非治国之府，实权贵之巢。

爵位不以功授，而以亲分；
官职不以能取，而以党得。
阿谀者进，直言者黜；
忠民者远，忠上者显。

律令繁而不行于权门，
刑罚峻而独施于布衣。
朝堂之上，空谈“人民”；
府库之中，尽入私囊。

名为公有，实为私藏天下；
号曰革命，实为更换旧主。

少者寒窗十载，
志在立身，而终无立锥之地；

壮者奔走终年，
力竭其劳，而不敢言家；

老者积劳一世，
筋骨既朽，而晚景凄凉。

失业遍野，生计如刃；
民心日寒，国本将倾。

而权贵子弟，
未尝稼穑，已据高位；
未识饥寒，反论天下；
凭一纸门第，夺百姓数代之积。

彼辈犹曰：“此乃安定。”
不知安定者，阶层永锢也。

更有甚者，
中共伪朝犹挟先贤之名，
以往昔功绩自免今日之罪。

然试问：

先贤之志，
欲使天下再有主仆乎？
抑欲使人民皆为其主？

革命之名，
可为子孙世袭之券乎？
抑当为天下共守之约？

若以历史遮羞，
是谓亵渎历史；
若以功劳免责，
是谓背叛革命。

是以我等不忍坐视，
遂举义旗，号曰自由军。

我军非为私怨而起，
非为权位而争；
乃为讨回被窃之誓言，
伸雪被弃之民意。

我军之所奉者，唯此三端：

人不以出身论贵贱，此谓平等；

民不以言论获其罪，此谓自由；

国不以权力践其民，此谓博爱。

除此之外，
不奉个人，
不尊家族，
不立不可问责之权威。

我军所志，
非复一朝一党之专断，
乃建真正之共和国：

使立法、行政、司法各守其职，
权相制而不相护；
使官由民举，亦可由民罢；
使法律在上，权力在下；
使言者无惧，民意可达。

此非妄想，
乃先贤未竟之志，
亦天下久困之解。

中共伪朝若犹执迷，
以封锁代改革，
以恐惧代民心，
以武力拒正义，

则天命去之，
非我军之罪；
民心离之，
非天下之过。

今特檄告四方：

权无制，则必腐；
政无民，则必亡；
革命若失其义，
终为历史所诛。

我军不惧死，
不恋位，
但求此役之后——

国为共和国，
法为万民法，
民为真正之主人。

谨此檄文，
告于天下有识之士。

——
自由军檄
